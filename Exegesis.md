# Exegesis

#### 框架注释

文档说明：

```c
/**
  ******************************************************************************
  * @file   
  * @brief  
  ******************************************************************************
  * @attention
  * 
  *  None
  *
  ******************************************************************************  
**/
```

引用；定义：

```
/* includes --------------------------------------------------------------------------*/
/* Private includes 非官方头文件声明----------------------------------------------------------*/
/* Exported types 类型定义------------------------------------------------------------*/
/* Exported constants 常量定义--------------------------------------------------------*/
/* Exported macro 宏定义------------------------------------------------------------*/
/* Exported functions prototypes 函数声明---------------------------------------------*/
```

分组：

```c
/**
 * @addtogroup 
 * @{
*/
......

 /**
  * @ }
 */
```

函数说明：

```c
 /**
  * @brief  
  * @param  
  * @retval 
  */
```
